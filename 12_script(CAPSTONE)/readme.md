#  Log Analysis 

A Python-based log processing pipeline for cleaning, analyzing, and summarizing 
traffic data utilizing CSV logs.

This project simulates a basic analysis workflow: 
ingest raw logs → clean and normalize data → analyze event patterns → generate security insights.

This serves as a "capstone" for me, as I've completed my Python class at my university. This is also
the end of this repo, but I will likely continue future projects in other repos.

## Features

- CSV log ingestion
- Data cleaning and validation
- Timestamp parsing
- Missing data imputation (`UNKNOWN`)
- Duplicate log removal
- Event frequency analysis
- IP activity tracking
- CLI-based execution using `argparse`
- Clean CSV export of processed logs
